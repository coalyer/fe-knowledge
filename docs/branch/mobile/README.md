Cordova 是用于使用 HTML，CSS 和 JS 构建移动应用的平台。

我们可以认为 Cordova 是一个容器，用于将我们的网络应用程序与本机移动功能连接。

默认情况下，Web 应用程序不能使用本机移动功能。

这就是 Cordova 进来的地方。它为网络应用和移动设备之间的连接提供了桥梁。

通过使用 cordova，我们可以使混合移动应用程序，可以使用摄像头，地理位置，文件系统和其他本地移动功能。
